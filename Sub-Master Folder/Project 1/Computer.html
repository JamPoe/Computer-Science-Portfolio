Computer Parts Explanation

Computer parts
  • Motherboard
  • CPU
  • RAM
  • HDD
  • Keyboard
  • Mouse
  • Monitor

    • Motherboard - The motherboard is the main circuit board that contains most of the components of a computer. It contains things like CPU.


    CPU - CPU is the Central Processing Unit. CPU processes the actions of a program by performing arithmetic operations needed to process the program.


    RAM - RAM is the Random-Access Memory. RAM is a form of computer data storage. RAM allows data items to be read and written in the same amount of time regardless of the order in which data items are accessed.


    HDD - HDD, or hard disk drive, is a data storage device used for storing digital information using rotating disks (called platters).


    Keyboard - A keyboard is the part of a computer in which input in the form of letters numbers and symbols can be inserted.


    Mouse - A mouse is a pointing devide used for navigation and movement in a GUI. Mouses help navigate through a two-dimensional surface.


    Monitor - A monitor is simply a visual display.
