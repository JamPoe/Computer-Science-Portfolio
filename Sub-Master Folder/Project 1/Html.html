    • HTML Parts Explanation

    HTML: HTML stands for Hyper Text Markup Language. HTML is used for building the basic structure of websites. HTML was created in 1989 by a man named Tim Berners-Lee. Just about all of html is written using tags, or elements.

    CSS: CSS stands for cascading style sheets. CSS is used to polish and refine a html file. Basically, this means that a website will look MUCH better when using CSS.     Tags and Elements: Tags are the main part of the HTML language. Tags are always enclosed in these (<>) symbols.

    Attributes: Attributes are extra parameters used in tags. An example of an attribute is "href" which allows you to write down a location or directory.

    Examples of tags:

    1. body - This tag is used to define the whole, or the "body" of an html.
    2. title - This tag labels the tab of the html, giving it a title.
    3. img - This tag enables you to implement photos into your html.
    4. audio - This tag is used to add audio into your html.
